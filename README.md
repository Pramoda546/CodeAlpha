# 🚀 CodeAlpha Frontend Development Internship Projects

Welcome to my **CodeAlpha Frontend Development Internship Repository**.
This repository contains the projects I completed as part of the **Frontend Developer Internship at CodeAlpha**, where I built multiple interactive web applications using modern frontend technologies.

The goal of these projects was to strengthen my understanding of **HTML, CSS, JavaScript, responsive design, and UI development** through practical implementation.

---

# 🛠️ Technologies Used

* **HTML5** – Structure and semantic layout
* **CSS3** – Styling, animations, and responsive design
* **JavaScript** – Interactivity and functionality

# 📂 Internship Tasks & Projects

## 🖼️ Task 1: Image Gallery

An interactive and responsive **Image Gallery** built using HTML, CSS, and JavaScript.

### Features

* Grid-based image gallery layout
* Next / Previous navigation buttons
* Lightbox image preview
* Hover effects and smooth transitions
* Fully responsive design

### Bonus Features

* Image filters or category sorting
* Enhanced animations

---

## 🧮 Task 2: Calculator

A functional **web-based calculator** that performs basic arithmetic operations.

### Features

* Clean calculator interface
* Arithmetic operations: **Addition, Subtraction, Multiplication, Division**
* Interactive buttons and display screen
* Real-time result calculation
* Clear/reset functionality

### Bonus Features

* Keyboard input support
* Improved UI styling

A **personal portfolio website** showcasing my skills, projects, and contact information.

### Sections Included

* Skills
* Projects

* Contact Information

### Features

* Clean and modern UI design
* Responsive layout for all devices
* Smooth scrolling navigation

## 🎵 Task 4: Music Player (JavaScript)

A **web-based music player** with interactive audio controls.

### Features

* Play / Pause functionality
* Next and Previous track controls

* Volume control

### Bonus Features
